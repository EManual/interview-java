JSP中的隐含对象是什么意思？有哪些隐含对象？

---------

JSP隐含对象是页面中的一些Java对象，JSP容器让这些Java对象可以为开发者所使用。开发者不用明确的声明就可以直接使用他们。JSP隐含对象也叫做预定义
变量。下面列出了JSP页面中的隐含对象：

  * application
  * page
  * request
  * response
  * session
  * exception
  * out
  * config
  * pageContext
